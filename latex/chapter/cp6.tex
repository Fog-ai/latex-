\section{总结与展望}

\subsection{论文总结}
随着物联网技术的快速发展，连接设备的数量呈指数增长，且这些设备广泛分布于现代社会各个领域。在诸如工业自动化、自动驾驶等时间敏感型应用中，控制中心的决策高度依赖于数据的新鲜度。传统的网络性能指标无法准确刻画这一特性，因此，AoI 作为衡量信息新鲜度的核心指标，被赋予了极高的研究价值。

本文立足于海量设备接入的大规模物联网通信场景，指出了传统随机接入机制中“盲目竞争”对网络 AoI 造成的恶化影响。秉承着“主动让渡与协同接入”的核心优化理念，本文设计了一种新的随机接入机制，并将其分别应用到无载波侦听与有载波侦听两种典型的无线接入环境中，设计出了等待 ALOHA 和等待 CSMA 两种协议，并构建严密的底层数学理论分析框架。本文的主要研究工作与创新成果总结如下：
\begin{enumerate}[wide]
    \item \textbf{提出等待随机接入机制}

    针对现有随机接入机制的局限性，本文提出了一种全新的\textbf{等待随机接入机制}。该机制的核心思想是正视信道碰撞的客观存在，在保留现有改进协议“成功传输后主动让步”的基础上，为节点在遭遇碰撞后引入了合理的等待与退避逻辑。通过引入这种确定的碰撞处理措施，本机制可打破高并发场景下的持续碰撞死锁，使网络在非理想的真实碰撞环境中依然能够维持较好的信息更新能力。

    \item \textbf{等待 ALOHA 协议的构建与分析}

    本章将所提出的等待机制首先应用于基础的等长时隙 ALOHA 框架，构建了等待 ALOHA 协议。我们通过马尔可夫链对系统进行建模，进而推导出了关于系统稳态工作点的非线性自洽方程，并证明了该方程解的存在性与唯一性。接着，我们还对等待 ALOHA 的性能进行了严密的分析，成功推导出了平均 AoI 与 AoI 违规概率的计算闭式。

    \item \textbf{等待 CSMA 协议的构建与分析}

    本章将等待机制进一步应用到了具备载波侦听与变长系统时隙的 CSMA 网络中，构建了等待 CSMA 协议。针对 CSMA 机制的特性，我们利用马尔可夫链对系统进行建模，并推导了考虑微时隙冻结特性的自洽方程，并对方程解的存在性与唯一性进行了证明。我们也在充分考虑变长时隙因素的基础上对它进行了性能分析，并最终得到了平均 AoI 与 AoI 违规概率的计算闭式。

    \item \textbf{完成了仿真验证并提炼了理论价值。}
    
    本文通过对比仿真证明了所提理论解析闭式的准确性，并验证了等待机制在真实非理想碰撞环境中的优越性。研究结果深刻揭示了“适度的主动退避不仅不会加剧信息老化，反而是提升系统整体时效性的有效途径”，为未来高并发场景下的 MAC 层协议设计提供了一种高度稳健的设计思路与理论依据。
\end{enumerate}

\subsection{研究展望}
本文主要聚焦于面向于 AoI 的随机接入机制设计与性能分析，虽取得了一定的成果，然而，受限于研究时间与模型的侧重点，本文的系统建模仍存在部分理想化假设。未来的研究工作可以从以下几个维度进行拓展与深化：
\begin{enumerate}[wide]
    \item \textbf{物理层信道模型的拓展。}本文考虑的是一个理想的信道条件，没有将路径损耗、信道衰减以及阴影效应等复杂物理层特性对数据包传输的影响考虑在内。在未来的工作中，将尝试引入更贴近真实无线传输环境的衰落信道模型，进一步提高所提算法的适用性。
    \item \textbf{排队机制与随机网络流量的引入。}本文的节点设计采用了按需生成模型且未设置数据缓存队列，从而剥离了数据到达率和服务率动态变化带来的额外影响。然而，实际通信系统中的业务流往往具备高度随机性。因此，未来的研究中将进一步结合排队论，构建带有节点队列、且支持随机数据到达与服务过程的系统级模型，探索其对网络整体 AoI 性能的影响。
    \item \textbf{系统参数的最优化求解与策略设计。}尽管本文中已经推导出了平均 AoI 与 AoI 违规概率的相关闭式，但在最优传输参数的理论求解与闭式最优化论证方面仍留有空白。下一步研究将重点探索上述理论表达式的极值条件，尝试构建并求解相关的优化问题，旨在为实际网络的参数配置提供具备数学支撑的最优策略。
\end{enumerate}
